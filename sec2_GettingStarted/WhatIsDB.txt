What Is A Database?
 - A Database is a collection of data.
 - A structured set of of computerized data with an accessible interface.
 - A method for accessing and manipulating data.
 - A relational DBs contains tables.

Relational Database Management System (RDBMS)
 - RDBMS is a way to access and connect the the database.
 - MySQL, PostgresSQL, Oracle Database, SQLite are example of popular (RDBMS) 
   that use SQL to interact with DBs.

MySQL vs SQL
 - SQL is the language/syntax we used when we 'talk' to the database.
 - SQL is a way we access and manipulate data.
 - MySQL (an RDBMS) uses SQL to interact with (or implement) the database.
